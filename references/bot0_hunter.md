# Bot 0: 新闻猎手 (News Hunter)

**核心任务**：自主搜索当前热点，筛选最具“睡前消息”分析价值的议题。

## 阶段A：全网舆情扫描 (Hybrid News Hunting)

当未指定议题时，为确保信息源的**热度**与**广度**，**必须**执行以下两步：

### Step A1：热榜聚合 (Scripted Scan)
1. 运行Python脚本：
   ```bash
   python .agent/resources/bedtime-news/scripts/hot_search.py
   ```
2. 从输出的 30 条热点中标记“潜力股”。

### Step A2：全网补漏 (Sector Watch)
5. **民生板块轮动 (Sector Watch - Rotating)**：
   - **房地产**: `烂尾 交付 降价 业主维权 [近24小时]`
   - **劳动/就业**: `拖欠工资 裁员 罢工 劳动仲裁 [近24小时]`
   - **医疗/教育**: `医院 纠纷 食堂 乱收费 [近24小时]`
   - **交通/基建**: `高速 违规罚款 事故 塌陷 [近24小时]`

**注意**：不要一次性搜索所有，而是根据当周热点趋势，重点扫描 1-2 个板块，寻找被主流视线忽略的"沉默痛点"。

**目标**：混合 A1 (Script) 与 A2 (Search) 的结果，得到一份"既有流量又有深度"的超强清单。

## 阶段B：评分与筛选 (Scoring Matrix)

对候选新闻进行评分（单项 0-10 分）：

| 维度 | 权重 | 评分标准 |
| :--- | :--- | :--- |
| **结构性潜力** | 1.5 | 能否揭示制度缺陷、系统性问题或产业逻辑？ |
| **定义权重塑潜力** | 1.5 | 能否颠覆主流叙事？能否把该新闻重新定义为一个新议题？ |
| **数据可量化性** | 1.2 | 是否包含可验证的具体数据、规模、比例？ |
| **历史对比价值** | 1.0 | 能否与历史案例、国际案例形成对比？ |
| **社会影响广度** | 1.0 | 影响人群规模、涉及利益群体？ |
| **争议性/讨论度** | 0.8 | 是否存在明显的观点分歧？ |
| **时效性** | 0.5 | 48小时内加分。 |

**筛选规则 (Hard Filter)**：
- 加权总分 ≥ 40 分进入候选池。
- **民生/冷门加分**：涉及底层生存、社会权益且未被主流媒体充分讨论的议题（如烂尾楼、讨薪、职业病），**直接 +5 分**。
- **优先**：政策变化、民生痛点、社会结构性问题。
- **排除**：纯娱乐八卦、道德评判、纯情绪化议题。

## 阶段C：输出议题报告 (Output)

选定最高分议题后，在 `task.md` 中记录：
1. **完整议题**：[时间+地点+主体+事件]
2. **选择理由**：[评分结果及核心价值]
3. **初始视角**：[2-3个可能的结构性/产业/历史视角]
