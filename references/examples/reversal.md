# 悬念反转式结构

## 适用场景
结论反常识，需要层层论证来颠覆读者预期。

## 结构模式
```
抛出反常识结论（引发好奇）
  → 解释为什么常识是错的
  → 层层论证新结论
  → 回扣开头，完成逻辑闭环
```

## 典型案例：高铁为什么不涨价

### 开头：抛出反常识结论
> "高铁票价十几年不涨，不是因为国家补贴，而是因为**涨不起**。"

读者预期：高铁是国家战略，不涨价是让利于民。
实际情况：完全相反。

### 解释常识为什么错
> "如果涨价能赚更多钱，国铁集团为什么不涨？"

- 国铁集团负债6万亿，每年利息支出就超过千亿
- 如果涨价能改善财务状况，没有任何理由不涨

**转折**："所以问题变成了：为什么涨价反而赚不到更多钱？"

### 层层论证
**第一层**：高铁的竞争对手不是飞机，是私家车和大巴
- 高铁涨价10%，一部分人就会选择自驾或大巴
- 尤其是价格敏感的群体（探亲、旅游）

**第二层**：高铁的成本结构决定了必须"薄利多销"
- 边际成本极低（多一个乘客几乎不增加成本）
- 固定成本极高（线路建设、维护、折旧）
- 最优策略是尽可能填满座位，而不是抬高票价

**第三层**：涨价会触发恶性循环
- 客流下降 → 收入减少 → 服务缩减 → 客流进一步下降

### 回扣开头
> "所以高铁不涨价，不是'惠民工程'，而是'别无选择'。"

**留钩子**："但这引出一个更大的问题：一个'涨不起价'的基础设施，如何消化6万亿债务？那就是另一期节目的话题了。"

## 转折句模板
- "但如果你以为这就是答案，那就太天真了。"
- "真相恰恰相反。"
- "听起来反常识，但逻辑是这样的——"
- "所以回到开头的问题，答案已经很清楚了。"
