# 层层剥皮式结构

## 适用场景
复杂问题需要逐步深入，每一层回答都引出更深的问题。

## 结构模式
```
问题A → 回答A，但引出问题B 
      → 回答B，但引出更深的C 
      → 回答C，最终指向本质
```

## 典型案例：垃圾处理悖论

### 第一层：表面矛盾
> 焚烧厂吃不饱 vs 溶洞塞满垃圾——为什么同时存在？

**回答**：地理错配。焚烧厂在城市，垃圾分散在农村。

**引出下一层**："但问题没这么简单——为什么农村垃圾到不了城市焚烧厂？"

### 第二层：机制断裂
**回答**："户集村收镇运县处理"模式在"镇转运"环节断裂，因为成本过高。

**引出下一层**："这就引出了下一个问题：为什么成本覆盖不了？钱从哪来？"

### 第三层：付费困境
**回答**：农村垃圾费收不上来，财政补贴压力大，转运服务缩水。

**引出下一层**："所以我们必须再往下挖一层：为什么城市焚烧厂不能主动接收农村垃圾？"

### 第四层：结构性障碍
**回答**：运输成本、行政边界、利益分配三重障碍。

**最终指向**：城乡公共服务供给能力的鸿沟 + 市场化基础设施无法覆盖低密度区域。

## 转折句模板
- "但问题没这么简单。"
- "这就引出了下一个问题："
- "所以我们必须再往下挖一层。"
- "到这里，真正的矛盾才浮出水面——"
