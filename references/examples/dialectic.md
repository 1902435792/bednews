# 对立统一式结构

## 适用场景
有明显正反观点的争议性话题，需要超越二元对立。

## 结构模式
```
正方观点（有道理）
  → 反方观点（也有道理）
  → 两边都漏掉了什么？
  → 真正的问题不在于X，而在于Y
```

## 典型案例：该不该发钱提振内需

### 正方观点
> "直接发钱是最高效的刺激消费方式"

- 美国疫情期间发钱，消费立竿见影
- 减少中间环节，钱直达居民手中
- 提振信心，打破"不敢消费"的恶性循环

**承认其合理性**："这个逻辑在短期内确实成立。"

### 反方观点
> "发钱会被储蓄吃掉，不如投资基建"

- 中国居民储蓄率高，发钱可能存起来
- 基建投资有乘数效应，带动就业
- 发钱是"一次性"的，基建是"长期资产"

**承认其合理性**："这个担忧在当前环境下也不是没有道理。"

### 两边都漏掉了什么？
> 争论的焦点是"发钱还是投基建"，但真正的问题是：**为什么居民不敢消费？**

- 正方假设：钱到手就会花
- 反方假设：投资能解决就业

**两个假设都回避了核心问题**：预期不稳定、社保不健全、收入增长乏力。

### 真正的问题
> 不在于"发不发钱"，而在于"如何修复居民对未来的信心"

发钱只是手段，真正需要的是：
1. 稳定的就业预期
2. 可靠的社会保障
3. 收入增长的可见通道

**结尾钩子**："这就是为什么发钱的争论永远没有结论——因为问题根本不在这里。"

## 转折句模板
- "这个观点有道理，但对面也不是没有道理。"
- "问题在于，双方都回避了一个更根本的问题。"
- "真正的矛盾不在于A还是B，而在于——"
- "所以争论发不发钱，其实是在回避真正的问题。"
