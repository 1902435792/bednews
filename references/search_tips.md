# 搜索技巧

## 选题阶段搜索（用户未指定议题时）

如果需要自行选择热点议题，优先使用以下策略：

| 搜索类型 | 搜索词模板 | 说明 |
|----------|------------|------|
| **热榜汇总** | `site:weibo.com 热搜榜 {当前年月} 热点事件` | 微博热搜是舆情风向标 |
| **舆情盘点** | `{当前年月} 舆情热点 社会议题` | 找专业舆情分析文章 |
| **权威新闻** | `site:thepaper.cn {当前年月} 热点` | 澎湃新闻的深度报道 |
| **争议事件** | `{当前年月} 争议 批评 反思` | 直接搜有争议的事件 |

**选题评估清单**：
- [ ] 是否有明确的"官民分歧"或"舆论撕裂"？
- [ ] 是否能找到至少3-5个关键数据？
- [ ] 是否涉及制度/结构性问题（不仅是个案）？
- [ ] 是否有"层层追问"的空间？

## 搜索模板

### 基础搜索组合
每个议题执行4-6次搜索，覆盖以下维度：

| 维度 | 搜索词模板 | 示例 |
|------|------------|------|
| **核心数据** | "[议题关键词] 数据 统计 {当前年份}" | "垃圾焚烧 产能 数据 2025" |
| **典型案例** | "[议题关键词] 案例 事件 曝光" | "溶洞 非法倾倒 垃圾 案例" |
| **政策背景** | "[议题关键词] 政策 规定 文件" | "农村垃圾处理 政策 标准" |
| **争议观点** | "[议题关键词] 争议 批评 问题" | "垃圾焚烧厂 选址 争议" |
| **成本/经济** | "[议题关键词] 成本 费用 价格" | "垃圾处理费 收费标准" |
| **对比参照** | "[议题关键词] 对比 国外 其他城市" | "日本 垃圾处理 模式" |

### 搜索词优化技巧

1. **加年份限定**: 使用**当前年份**获取最新数据（不要照抄示例中的年份）
2. **加权威来源**: 添加 "统计局" "官方" "报告" 提高可信度
3. **用反向搜索**: 搜索反对意见以获得多元视角
4. **用具体数字**: 如找到一个数字后，搜索该数字验证

### 数据验证

搜索到数据后，执行以下验证：
- [ ] 数据来源是否明确？
- [ ] 是否有多个来源交叉印证？
- [ ] 数据时效性如何？
- [ ] 是否有明显的利益相关偏向？

## 上位概念搜索（历史镜像增强）

当直接搜索议题关键词未命中历史节目时，**必须**尝试以下策略：

### 1. 抽象上位概念
将具体议题抽象为更高层次的概念：

| 具体议题 | 上位概念搜索词 |
|----------|----------------|
| 死了么APP | "死"、"人文"、"殡葬"、"独居" |
| 知识付费 | "教育"、"焦虑"、"消费" |
| 长租公寓爆雷 | "资本"、"租房"、"金融化" |
| 外卖骑手困境 | "零工"、"劳动"、"平台" |

### 2. 方法论/批判视角
从分析框架角度搜索类似逻辑的历史案例：

| 方法论视角 | 搜索词 | 适用场景 |
|------------|--------|----------|
| 资本批判 | "资本"、"马克思"、"剥削" | 涉及商业模式/利益分配 |
| 消费主义 | "消费"、"符号"、"焦虑" | 涉及用消费代替行动 |
| 制度空白 | "法律"、"监管"、"灰色地带" | 涉及野蛮生长的行业 |
| 技术异化 | "技术"、"平台"、"算法" | 涉及互联网/AI问题 |

### 3. 搜索执行示例
```bash
# 以"死了么APP"为例，命中率优化策略：
python search_index.py "死"           # 单字抽象
python search_index.py "人文"         # 上位概念
python search_index.py "焦虑"         # 情绪视角
python search_index.py "资本"         # 批判视角
python search_index.py "消费主义"     # 方法论
```

## 理论备选库（经验补充视角）

当数据搜索和历史镜像都未能提供足够支撑时，可调用以下理论框架作为分析工具：

### 经典理论框架

| 理论来源 | 核心概念 | 适用场景 | 引用示例 |
|----------|----------|----------|----------|
| **马克思** | 异化劳动、商品拜物教 | 劳动者权益、消费主义 | "把XX也异化成了商品" |
| **韦伯** | 工具理性、科层制 | 官僚体系、效率至上 | "理性化的囚笼" |
| **鲍德里亚** | 符号消费、拟像 | 虚假需求、消费社会 | "消费的是符号而非实物" |
| **福柯** | 规训、全景敞视 | 监控、权力运作 | "无处不在的凝视" |
| **毛泽东** | 矛盾论、实践论 | 制度分析、群众路线 | "主要矛盾和次要矛盾" |

### 使用规则
1. **理论是工具，不是装饰**——只在确实能增强论证时引用
2. **用白话解释**——不堆砌术语，用读者能懂的方式表达
3. **标注来源层级**——在论证链条中标注 `[理论框架]` 以区别于 `[搜索验证]`

## "香农信息论"关键词选择

搜索关键词应选**信息量最大**（最独特、出现概率最低）的词：

| 原则 | 说明 | 示例 |
|------|------|------|
| **独特性 > 正确性** | 独特词即使略有错误也比通用词有效 | ✅ "马督工" vs ❌ "马前卒" |
| **提取专有名词** | 从原始信息中提取最独特的名称、数字 | ✅ "水司楼 独山" vs ❌ "贫困县 债务" |
| **引语片段搜索** | 用原文中的独特表述作为搜索词 | ✅ "人力资源金融化" |
| **组合独特词** | 两个独特词组合比一个通用词更有效 | ✅ "龙山惨案 2009 强拆" |

## "找茬式"对比搜索

当同一事件有多个信源报道时，刻意对比不同级别信源：

**操作模板**：
```
搜索1: [事件] site:xinhuanet.com    ← 官方信源（本钱大，措辞谨慎）
搜索2: [事件] site:thepaper.cn      ← 主流媒体（有品牌约束）
搜索3: [事件] 争议 评论 自媒体      ← 民间视角（本钱小，可能极端）
```

**分析重点**：
- 不同信源对**同一数字**的引用是否一致？
- 官方通报与媒体报道在**关键措辞**上有无差异？
- **时间线**是否一致——谁先报、谁后报、谁删改过？
- 偏差方向是否与信源"本钱"相关？（详见 logic_appendix.md §6）
